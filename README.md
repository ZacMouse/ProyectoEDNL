# Electro Tecnologías — Sistema de Reparaciones

Aplicación de escritorio desarrollada en Python con Tkinter para la
gestión de reparaciones de dispositivos electrónicos. Permite registrar
equipos, asignar técnicos y generar reportes de manera sencilla mediante
una interfaz gráfica moderna.

------------------------------------------------------------------------

CARACTERÍSTICAS

-   Registro de dispositivos para reparación
-   Asignación de técnicos a trabajos
-   Generación de reportes
-   Interfaz gráfica amigable (Tkinter)
-   Arquitectura modular

------------------------------------------------------------------------

views/ - app.py - estilos.py - vista_registro.py - vista_asignacion.py -
